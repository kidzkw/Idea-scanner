## Core scan

Query SEC EDGAR or home-country register based on `mode`:

### mode=create_tracker (first-time setup)
- WebSearch `"<TICKER> SEC EDGAR CIK"` → get CIK
- WebFetch `https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=<CIK>&type=&dateb=&owner=include&count=40`
- If 403 → fallback protocol (§5c.6):
  1. WebSearch `"<TICKER> 8-K 2026 site:sec.gov"`
  2. WebSearch `"<TICKER> earnings press release 2026"` (IR page)
  3. WebFetch `https://efts.sec.gov/LATEST/search-index?q=<query>&forms=<form>`
  4. If all fail → ask user to paste EDGAR output manually
- Create `03_tickers/SEC/<TICKER>_sec.md` with 7 required fields (§5c.4)
- Append row to `_SEC_INDEX.md`

### mode=daily_scan (active trackers, 48h lookback)
- For all active trackers in `_SEC_INDEX.md`, fetch last 48h filings
- Flag any new Form 4 / 8-K Item 1.01 / SC 13G/D → generate new raw to `02_raw/_inbox/`

### mode=t_minus_1 (earnings eve)

- Refresh tracker with latest filings

### mode=t_plus_0 (earnings day)
- Fetch actual earnings 8-K/6-K content

### mode=t_plus_3 (follow-on)

### mode=t_plus_45 (10-Q due date, US only)

- Cross-check vs earnings press release numbers
- **Mismatch → write alert to `_AUDITS/YYYY-MM-DD_10q_press_mismatch_<TICKER>.md`**

### mode=event_driven

- Any hit → new raw to `02_raw/_inbox/`

## Source weight

SEC filings = **Weight 100** (Source Weight Ladder §4.3) — top of hierarchy. When SSOT Thesis Log cites a number, prefer SEC filing citation over sell-side / media.

## Insider alert thresholds

- Form 4 buy/sell ≥$1M → flag in `_SEC_INDEX.md` insider alert section
- Form 4 CEO/CFO transaction regardless of size → flag
- SC 13G/D new ≥5% filer → high priority (e.g., AMAT holding 9% of BESI is M&A signal)
- 10b5-1 plan filings → track with adoption date vs execution date

## Downstream triggers

- New Form 4 ≥$1M → generate raw `02_raw/_inbox/YYYY-MM-DD_<TICKER>_sec_form4-<insider>.md` → emergency lane to 调度员
- 10-Q ≠ press release numbers → `_AUDITS/` entry + main thread notifies user
- New 8-K M&A Item 1.01 → trigger SSOT §四 Catalyst update (main thread dispatches `研究员`)
- Earnings 6-K/8-K → update `05_triggers/catalyst_calendar.md` with result row
