## Embedded rules digest

**Filename + frontmatter authority**: `00_rules/FILENAME_FORMAT_v1.md` (v1.0, 2026-04-25 总监决策)
- Filename pattern: `<YYYY-MM-DD>_<TICKER>_<SOURCE>_<topic-kebab-case>.md`

**Required frontmatter fields** (per FILENAME_FORMAT_v1.md §3):

**Grok truncation patterns** (§1.4 Stage 1):
- `...` (ellipsis)

**Rule 1 China ADR reject** (memory/feedback_no_china_adrs):

## Inputs

- `raw_path`: absolute path to raw file in `02_raw/_inbox/<filename>.md`

## Core protocol

### Step 1 — Read frontmatter

```python
content = Read(raw_path, limit=50)  # cheap, just frontmatter
parse YAML between --- markers
```

### Step 2 — Required field check

```python
required = [source, date, ticker, source_detail, topic, confidence, status]
for field in required:
    if not present or empty:
        return {status: rejected, reason: frontmatter_missing:<field>}
```

### Step 3 — Rule 1 China ADR

```python
china_list = {TCEHY, BABA, BIDU, JD, PDD, NTES, BILI, NIO, XPEV, LI, FUTU, TME, ZH, KE, ...}
if ticker in china_list OR ticker.endswith(('.HK', '.SS', '.SZ')):
    return {status: rejected, reason: rule1_china_adr, ticker: <T>}
```

For ambiguous cases (Chinese-founded US listing, e.g. HESAI): WebSearch optional — but since Triage must be cheap, default to accept and let user override.

### Step 4 — Rule 2 ETF

```python
etf_list = {SMH, SOXX, NVDL, AIQ, BOTZ, QQQ, SPY, ...}
if ticker in etf_list:
    return {status: rejected, reason: rule2_etf, ticker: <T>}
name_hint = raw.source_detail + raw.topic + raw.url
if any(kw in name_hint for kw in ['ETF', 'index fund', 'ETN', 'UCITS']):
    return {status: rejected, reason: rule2_etf_hint}
```

### Step 5 — Grok truncation (if source=grok)

```python
if raw.source == 'grok':
    body = Read(raw_path)  # full read
    truncation_patterns = ['...', '(additional', '(15+ more', 'and more')
    for p in truncation_patterns:
        if p in body:
            # not reject — flag for re-fetch
            return {status: accepted, flags: [truncation_detected], 
                    update_frontmatter: {data_quality: partial}}
```

### Step 6 — Accept

```python
if all checks pass:
    return {status: accepted, flags: [], ticker: <T>, source_weight: <from frontmatter>}
```
