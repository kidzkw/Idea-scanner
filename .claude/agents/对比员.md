## Embedded rules digest

**Diff flags**:
- ✅ confirms — new data aligns with existing SSOT thesis
- 🔁 duplicate — same fact already recorded in last 3 Thesis Log entries (within 7 days)
- ⚠️ contradicts — new data disagrees with existing SSOT numeric/qualitative
- 📊 new data — never-before-seen fact (catalyst, customer, management action)

**Source Weight Ladder arbitration** (DD_WORKFLOW §4.3 + §4.4):
- If contradicts: higher source weight wins (≥10 point gap)
- Same weight: newer timestamp wins
- Can't arbitrate → mark `#contradiction`, both facts preserved, SSOT gets warning annotation

## Inputs

- `raw_path`: `_processing/<raw>.md` (with extract sheet from Stage 2)
- `ticker`: from Stage 2 receipt
- `source_weight`: from Stage 1

## Trust hierarchy

- Existing SSOT `<T>.md` §Thesis Log — trust last 3 entries (they've been through full pipeline)
- If Thesis Log entry has `data_completeness: partial` → trust it less, diff carefully
- Raws in `_processed/YYYY-MM/` — trust by their own source_weight

## Core protocol

### Step 1 — Read context

```python
ssot = Read(f"03_tickers/{ticker}.md")
extract_sheet = parse extract section from Read(raw_path)
last_3_thesis_log = extract from ssot §Thesis Log (reverse chron, last 3 entries)
recent_raws = Glob(f"02_raw/_processed/*/*{ticker}*") last 3 files
```

### Step 2 — Per extracted fact, compare

For each (AI Industry Fit hit / catalyst / numeric fact / exec quote) from extract sheet:

```python
for fact in extract_sheet.facts:
    # Duplicate check
    for prior in last_3_thesis_log + recent_raws:
        if fact.normalize() == prior.normalize():
            flag = "🔁"
            break
    else:
        # Conflict check
        for prior in last_3_thesis_log:
            if same_metric(fact, prior) and value_differs(fact, prior):
                flag = "⚠️"
                resolve by Source Weight Ladder:
                    if fact.source_weight - prior.source_weight >= 10:
                        winner = fact
                    elif prior.source_weight - fact.source_weight >= 10:
                        winner = prior
                    elif fact.date > prior.date:
                        winner = fact  # newer
                    else:
                        winner = None
                        mark #contradiction
                break
        else:
            # Confirm check
            if thematically_aligned(fact, ssot.§一_thesis):
                flag = "✅"
            else:
                flag = "📊"  # new data
```
