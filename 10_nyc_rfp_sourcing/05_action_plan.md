# 90-Day Action Plan — G0 → G4

> Goal: in 90 days, the user has a registered entity, an MBE application submitted,
> a PASSPort account live with commodity codes, a 1-page capability statement,
> and ≥5 agency MWBE liaisons contacted. **Not** a first PO — that requires the
> MBE cert to clear (90–120 days from submission).

---

## Week 1 — Decide & document

- [ ] Read all files in this project; answer the 5 gate questions in `01_profile_qualifications.md`
- [ ] If gate fails (visa status, PNW cap, ancestry doc gap), STOP and re-scope
- [ ] Choose entity structure with NY attorney (LLC vs S-corp; default LLC for first round)
- [ ] Pick the 2 anchor categories from `03_target_categories.md` (recommended: janitorial + non-medical PPE)

## Week 2 — Entity formation

- [ ] File NY LLC Articles of Organization (Division of Corporations)
- [ ] Obtain EIN (IRS Form SS-4, online, same-day)
- [ ] Open business bank account (need EIN + filed Articles)
- [ ] File NY Publication requirement (LLC must publish in 2 newspapers, 6 weeks — start now)
- [ ] Register with NY Department of Taxation (Form DTF-17 + ST-120 resale cert)
- [ ] Lease business address (or use registered agent service if home address inadequate)
- [ ] Draft Operating Agreement showing 51%+ ownership by user

## Week 3 — Compliance baseline

- [ ] Get general liability insurance quote ($1M, target $2K–$4K/yr premium)
- [ ] Get workers' comp policy (NY State Insurance Fund if no employees yet → minimum premium)
- [ ] Quote customs bond ($50K bond ≈ $500–800/yr)
- [ ] Identify customs broker — interview 2; pick one
- [ ] Open a freight forwarder account (Flexport, Freightos, or local forwarder)

## Week 4 — China sourcing intelligence

- [ ] For each anchor category, identify 3 factory candidates via Alibaba / Made-in-China / 1688 / trade-show contacts
- [ ] Request quotes on 5 representative SKUs per category (FOB Shanghai/Shenzhen)
- [ ] Verify factory: business license + export license + audit report (SMETA/BSCI) + sample
- [ ] Run UFLPA traceability check on any cotton/aluminum input
- [ ] Build a landed-cost spreadsheet: FOB × 1.25 (Sec 301) × 1.08 (freight+brokerage) = landed

## Week 5 — Benchmark NYC pricing

- [ ] Pull GSA Advantage prices for each of 5 SKUs (federal benchmark)
- [ ] Pull NYS OGS Centralized Contracts prices (state benchmark)
- [ ] Search PASSPort for past awards in target categories (look up "EPIN" + agency on CROL)
- [ ] Calculate gross margin at three price points (low/mid/high). Target ≥35% gross.

## Week 6 — MBE application prep

- [ ] Assemble all documents per `01_profile_qualifications.md` § 6
- [ ] Draft Personal Net Worth statement (assets - liabilities, ex-house, ex-business)
- [ ] Draft business narrative (1 page: who, what, where, when, why)
- [ ] Schedule SBS pre-submission workshop (free, virtual)

## Week 7 — Submit MBE application

- [ ] Submit via `nyc.gov/mwbe` portal
- [ ] Note ticket #; expect 90–120 day review
- [ ] Simultaneously submit **NYS MWBE** via [esd.ny.gov/mwbe](https://esd.ny.gov/mwbe-new-certification) (parallel; different portal)

## Week 8 — PASSPort enrollment

- [ ] Register vendor account at PASSPort
- [ ] Complete vendor questionnaire (financials, prior contracts if any)
- [ ] Select commodity codes — for janitorial + PPE, enroll in NIGP codes covering:
  - 485 (Janitorial Supplies — General)
  - 200 (Bags, Bagging Materials)
  - 345 (Cleaning Compounds & Solvents)
  - 080-83 (Personal Safety Equipment)
  - 200 (Bags & Bagging)
  - 175 (Clothing — Work, Industrial)
  - (Get full NIGP code list from PASSPort enrollment wizard)
- [ ] Sign up for **email alerts** on those commodity codes

## Week 9 — Capability statement & marketing

- [ ] Draft 1-page Capability Statement (PDF) — include:
  - Core competencies (the 2 anchors + 1 adjacency)
  - Differentiators (China direct sourcing, MBE-certified pending, NY-based, EIN, DUNS/UEI)
  - Past performance (or "newly formed, principal experience in …")
  - Codes — NAICS, NIGP, SIC, UEI
  - Contact info (one person, one phone, one email)
- [ ] Get a DUNS / UEI (via SAM.gov; free; needed for some PASSPort fields and for any federal cross-listing)
- [ ] Build simple website (landing page is enough; agency buyers do click through)

## Week 10 — Agency outreach round 1

- [ ] Email cap statement to ≥5 MWBE liaisons:
  - NYPD `mwbe@nypd.org`
  - DCAS — find liaison via DCAS site
  - DOE — Office of Pupil Transportation, School Food, Division of Procurement (3 separate desks)
  - DSNY — Office of Sustainable Procurement
  - Parks — agency procurement office
- [ ] Use a CRM (HubSpot free tier, Airtable, or just a Google Sheet) to track follow-ups
- [ ] Set 2-week follow-up reminders

## Week 11 — PASSPort scanning + first quote attempt

- [ ] Daily scan of PASSPort + City Record Online for solicitations in commodity codes (see `06_data_sources.md` for the recipe)
- [ ] If any small-purchase quote request lands (cert-pending vendors *can* be quoted in some cases) — respond inside 48h
- [ ] Score every RFP via `opportunities/_TEMPLATE.md` before responding; pursue only ≥7/10

## Week 12 — Reflect + adjust

- [ ] Review CRM: how many of 5 liaisons replied? Whose feedback was substantive?
- [ ] Refine cap statement based on questions agencies asked
- [ ] Identify the 2nd cohort of 5 agencies to email
- [ ] Check NYC MBE application status; if delayed beyond 120 days, escalate via SBS support
- [ ] Decision point: if zero engagement after 60 days of outreach, re-pick categories

## Tripwires (any single one → pause + reassess)

1. NYC MBE application denied or in "deficiency" past day 150
2. UFLPA detention on first textile shipment (don't ship cotton without trace docs)
3. First 5 agency outreach emails: 0 responses
4. Cert cost on chosen SKUs exceeds 10% of expected year-1 category revenue
5. Working capital availability falls below 30% of pipeline PO sum
